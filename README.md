# Loan Approval Prediction System

## Overview
The Loan Approval Prediction System is a web application developed using Python and Flask that predicts whether a loan application is likely to be approved based on user-provided financial information. The project demonstrates the application of machine learning in financial decision-making.

## Features
- User-friendly web interface
- Loan approval prediction
- Data preprocessing and model training
- Real-time prediction results
- Responsive design using HTML and CSS

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML
- CSS

## Project Structure

## How to Run

1. Clone the repository.
2. Install the required Python libraries.
3. Run the application:

```bash
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
